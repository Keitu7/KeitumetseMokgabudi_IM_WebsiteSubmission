Data Art

Bar Chart - GLOBAL TEMPERATURES

Using three-toned colour schemes (colours Yellow, Orange, and Red) illustrates how each shade of each colour demonstrates a progression of the units. The raw data represents the changes in the Earth's global temperature throughout the years. Each shade/stripe of the colours Yellow, Orange, and Red, represents the average temperature for a single year (University of Reading, 2023).
